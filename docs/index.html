Warum sollte man Backups machen?

Was ist ein Datenverlust?

  • Hardwarefehler (Getränk auf Notebook)
  • Softwarefehler
  • Benutzerfehler
  • Diebstahl (Mobile entwendet)
  • Brand
  • Wasserschaden (Mobile im Meer)
  • Ransomware

Wie kann man Datenverlust verhindern?

  • Backups machen
  • Verschlüsselung
  • Regelmäßige Updates

Wer ist verantwortlich?

Jeder allein ist verantwortlich für seine Daten!

Niemand sonst wird es für dich tun.

  • Es ist deine Aufgabe, deine Daten zu schützen
  • Es ist deine Aufgabe, deine Daten zu sichern
  • Es ist deine Aufgabe, deine Daten zu verschlüsseln
  • Es ist deine Aufgabe, deine Daten zu löschen

Wann sollte man Backups machen?

Regelmäßig und vor Änderungen an den Daten.

Was ist ein Backup?

Definition

Ein Backup ist eine Kopie von Daten, die an einem anderen Ort gespeichert wird, um sie im Falle eines Datenverlusts wiederherstellen zu können.

Arten von Backups

  • Vollbackup
  • Inkrementelles Backup
  • Differenzielles Backup
  • Versioniertes Backup

Backup-Strategien

  • 3-2-1 Regel
  • Grandfather-Father-Son GFS
  • Offsite Backup
  • Cloud Backup

Quiz Runde

Frage 1: Was ist die 3-2-1 Regel?

Antwort 1

  • 3 Kopien der Daten (1 Original + 2 Backups)
  • 2 verschiedene Medien
  • 1 Kopie an einem anderen Ort

Frage 2: Ist die Sicherung auf einem NAS ein Backup?

Antwort 2 - Nein, ein NAS Backup ist kein vollständiges Backup

  • Es kann Kopien von Daten enthalten
  • Es schützt vor Datenverlust durch Hardwarefehler
  • Es ist meinst keine Kopie an einem anderen Ort
  • Es schützt nicht vor Ransomware
  • Es schützt nicht vor Diebstahl

Frage 3: Ist die Sicherung von Fotos auf SD-Karten ein Backup?

Antwort 3 - Nein, ein SD-Karten Backup ist kein vollständiges Backup

  • Taugt nicht als Langzeitarchivierung (10 Jahre oder weniger)
  • Es schützt nicht vor Datenverlust durch Hardwarefehler
  • Es sind nicht unterschiedliche Medien (wer hat noch ein Diskettenlaufwerk?)
